Tela Azul da Morte

Essa é pra quem usa Windows 7 ou 8. Se você é adepto do Windows provavelmente já ouviu falar da famigerada “Tela Azul da Morte”, bom se ela acontecer com você é um sinal de que o sistema operacional desligou para evitar que problemas mais graves aconteçam com seu computador, uma espécie de mecanismo de defesa do sistema. Isso ocorre quando o sistema identifica um erro do qual não pode se recuperar sem que haja perda de dados, detecta também pontos do sistema que foram corrompidos bem como falha no hardware, que é o caso mais grave.

Analizando a Tela Azul

Já que a tela azul pode significar vários problemas, neste tutorial vamos nos deter apenas a análise da tela azul propriamente dita, visto que ela contém informações muito importantes para o diagnostico do problema. Para isto no Windows 7, após as duas primeiras linhas que dizem “Foi detectado um problema e o Windows foi desligado para evitar danos ao computador”, em alguns computadores essa mesma mensagem aparece em inglês, você verá a descrição do que está ocasionando problema no seu Windows. Abaixo da descrição do problema a tela também exibe algumas instruções que podem ser bem úteis na resolução do problema. Já as últimas duas linhas são técnicas e dão dicas para você saber mais especificamente o que foi detectado, esse código de erro é essencial na hora de buscar uma solução. Pesquisando esse código de erro na internet você certamente encontrará as causas e possíveis formas de resolver o seu problema especifico, mas abaixo encontrasse uma pequena lista de erros mais comuns apontados pela tela azul.

Erros Mais Comuns

Abaixo segue uma pequena lista com os erros e códigos mais comuns apresentados pela tela azul.

PAGE_FAULT_IN_NONPAGED_AREA (STOP 0x00000050)

Nesse código a tela azul aparece porque um driver ( normalmente um instalado recentemente) está a tentar gravar na memória somente leitura. A melhor aposta para corrigir este erro é para reverter o sistema para uma data anterior, antes que o driver foi desinstalado. Você também pode remover o driver instalado recentemente.

IRQL_NOT_LESS_OR_EQUAL (STOP 0x0000000A)

Existem inúmeras causas para esse erro, no entanto na maioria das vezes o erro pode ser atribuído a uma falha no software ou uma instância de hardware defeituoso. Se um driver específico está listado na tela azul, reinicie o computador e remova esse driver específico. Se o erro em vez inclui uma peça de hardware verifique a integridade do mesmo e caso seja necessário efetue a troca.

KERNEL_MODE_EXCEPTION_NOT_HANDLED (STOP 0x0000008E)

Esse erro é mais comumente o resultado de um vírus ou software de terceiros mal-intencionados. Isto é quase certamente o caso se o erro lista o arquivo Win32k.sys quando a tela azul aparece. Você deve executar o seu antivírus ou antispyware, afim de elimina-los.

Inacessível Boot Device (STOP 0x0000007B)

Este erro na maioria das vezes ocorre durante a inicialização do Windows, logo após a tela inicial aparece. Ele indica que o sistema operacional Windows não é capaz de recuperar dados de um sistema ou partição de boot. Os arquivos de sistema corrompidos também podem causar este problema. Você pode precisar restaurar ou reinstalar o sistema operacional.
